    Detekt

    For Kotlin Multiplatform projects Detekt (v1.23.8) won't automatically exclude build folders from scanning (and end up failing with incorrect style for generated code ü§®). The solution is to exclude it manually:

    build.gradle.kts
    1
2
3
4
5
    tasks.withType<Detekt>().configureEach {
    exclude { element ->
        element.file.path.contains("/build/generated/")
    }
}
